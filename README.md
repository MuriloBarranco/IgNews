# Ignews

Esse projeto foi desenvolvido no curso ignite da Rocketseat e tinha como objetivo criar uma blog usando tecnologias como Next, Stripe, Prismic CMS.

# 👩‍💻Tecnologias

- Typescript/Javascript
- Next
- Scss
- Prismic
- Stripe

# Como executar

    1 Clone o repositorio
    2 yarn
    3 yarn dev
